table: utilities for operations with tables

Provides convenience functions for operating with tables, including combine_namespaces() for combining ->-separated namespaces of values or chemicals into a single namespace; and combine_columns() for combining individual columns into a single column.

Code author: Peter Kraus

Functions

combine_namespaces(a, b[, conflicts, ...])

Combines two namespaces into one.

combine_columns(a, b[, fillnan, output])

Combines two columns into one.
